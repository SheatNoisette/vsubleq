# VSubleq

Simple subleq interpreter made for fun in V.

# Building

You need a V compiler to build VSubleq
```
$ v .
```
It will build `vsubleq` executable in the repository folder.

# Example

```
$ ./vsubleq -f hello.sq
Hello, world!
```
